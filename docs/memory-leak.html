非协程泄漏导致的内存泄漏

  1. 子字符串：s0 = s1[:50]（s0是全局变量，s1是临时变量）
    • 正确做法：s0 = (strings.Builder{}).WriteString(s1[:50])
  2. 子切片： s0 = s1[:50]（s0是全局变量，s1是临时变量）
    • 正确做法：copy(s0, s1[:50])
  3. 在s1[1:3]前未重置切片内的指针元素
  4. time.Ticker不再使用但是没有停止它
  5. time.Ticker在for循环中不断创建，见go-demos/leak-timeTicker
  6. 延迟调用导致的暂时性内存泄露，见链接
